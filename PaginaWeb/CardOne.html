Crear un Share Sheet para poder compartir en la aplicación

Hoy veremos como presentar un Share Sheet para compartir, también conocido como controlador de vista de actividad de la interfaz de usuario en su aplicación

Una vez cumpla con todo los requisitos, ¡creeamos nuestro proyecto Xcode! Abra Xcode y,en el modal que aparece,seleccione Create a New Xocode Proyect. Vamos a crear una aplicacion IOS, asi que en la pestaña IOS, seleccione la opcion Aplicacion.

Crea tu botón

Lo primero que vamos a hacer es añadir nuestro boton de compartir, cree su botón usando el botón Compartir de SF Symbols y como adicional le agregamos un titulo y un color. 


  var body: some View {
    Button(action: shareButton) {
        Image(systemName: "square.and.arrow.up")
            .font(.title)
            .foregroundColor(.red)
        
    }
}

Crear la función

Lo segundo es crear la función para mostrar u ocultar su Share Sheet cuando el usuario haga clic en el botón de compartir. 


        func shareButton() {        
          let url = URL(string: "https://www.apple.com/la/")
          let data = UIActivityViewController(activityItems: [url!], applicationActivities: nil)

          UIApplication.shared.windows.first?.rootViewController!.present(data, animated: true, completion: nil)
          
      }

Codigo Completo

Finalmente, ¡Es tan simple como eso! Ahora, cuando un usuario haga clic en el botón Compartir, se mostrará una hoja para compartir, entonces el código final debería verse como la imagen que se encuentra abajo. 


   struct ContentView: View {
       var body: some View {
          Button(action: shareButton) {
                Image(systemName: "square.and.arrow.up")
                    .font(.title)
                    .foregroundColor(.red)
                
            }
        }
        
        func shareButton() {
            let url = URL(string: "https://www.apple.com/la/")
            let data = UIActivityViewController(activityItems: [url!], applicationActivities: nil)
            
            UIApplication.shared.windows.first?.rootViewController!.present(data, animated: true, completion: nil)
            
        }
        
    }
