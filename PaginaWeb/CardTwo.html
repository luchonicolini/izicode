Crear Tarjetas de imagen usando ScrollView

El dia de hoy crearemos una app que contiene un modelo de datos que va a iterar la información de nuestra tarjeta medainte un scrollview.

Una vez cumpla con todo los requisitos, Abra Xcode y, en el modal que aparece, seleccione Create a New Xocode Proyect. Vamos a crear una aplicacion IOS, asi que en la pestaña IOS, seleccione la opcion Aplicacion.

Modelo de Datos

Lo primero que vamos a hacer es crear nuestro primer Modelo de Datos, para eso creamos un nuevo archivo y lo llamaremos Model. Un modelo es una estructura que representa las variables que debe contener un tipo de datos. Al modelo lo llamaré Company cada instancia de Company debe tener una identificación, un nombre con una imagen y un eslogan. Ahora se ajustará a un protocolo Identifiable , de modo que cada Company pueda identificarse de forma única por su ID 


  struct Company: Identifiable {
    var id = UUID()
    var name: String
    var image: String
    let slogan: String
}

Agregar datos dentro de Company

Una ves que se crea el modelo de datos, crearemos una matriz de instacias de Company. dado que el ID es un UUID, solo necesitamos proporcionar un nombre, una imagen y el eslogan. Ahora dado que la variable companies es una matriz, podemos recorrela en nuestra Vista. 


    var companies: [Company] = [
    Company(name: "Movistar", image: "img1", slogan: "Compartida la vida es más"),
    Company(name: "Personal", image: "img2", slogan: "Hagamos que todo suceda"),
    Company(name: "Tigo", image: "img3", slogan: "Aquí estoy contigo a sonrie, tienes tigo")
    
]

Vamos a crear nuestro CategoryItem

Vamos a crear un archivo nuevo y lo llamaremos CategoryItem. Dentro de CategoryItem vamos a diseñar nuestra tarjeta. lo primero es añadir var company: Company afuera de nuestro body. 


 
  struct CategoryItem: View {
    var company: Company

    var body: some View {
        //..

Ahora la vista previa no funcionara por lo que debe pasar el primer elemento de la matriz companies 


        struct CategoryItem_Previews: PreviewProvider {
          static var previews: some View {
              CategoryItem(company: companies[0])
          }
      }

Agregamos dentro del cuerpo de CategoryItem un VStack() que contiene otro VStack(), Reemplace la etiqueta del enlace (el texto hello world!) por el siguente codigo. 


    
        struct CategoryItem: View {
          var company: Company
          
          var body: some View {
              VStack(alignment: .leading, spacing: 16.0) {
                  Image(company.image)
                      .resizable()
                      .renderingMode(.original)
                      .aspectRatio(contentMode: .fit)
                      .cornerRadius(15)
                      .shadow(radius: 3)
                  
                  VStack(alignment: .leading) {
                      Text(company.name)
                          .font(.title)
                          .foregroundColor(.primary)
                      Text(company.slogan)
                          .font(.headline)
                          .foregroundColor(.secondary)
                  
                  }
                  
                  .offset(y: -28)
                  .padding()
              }
              .padding()
          }
      }

Crear nuestro HomeView y añadimos el Contenido de nuestas tarjetas

Una vez creado el archivo HomeView , le agregaremos un NavigationView y un ScrollView que contiene un VStack 


    struct HomeView: View {
      var body: some View {
          NavigationView {
              ScrollView(.vertical, showsIndicators: false) {
                  VStack(spacing: 0) {
                     
                  }
              } 
          }
      }
  }

Diseño de ScrollView

Debajo del VStack en el contenido, recorreremos los paquetes de datos y mostraremos el contenido de CategoryItem para cada elemento. 


  VStack(spacing: 0) {
    ForEach(companies) { data in
       CategoryItem(company: data)
        
    }
}

Completemos nuestro HomeView

Finalmente, le agregarmos un background tasparente y un navigationBarTitle al cual llamaremos Compañías telefónicas. 


    struct HomeView: View {
      var body: some View {
          NavigationView {
              ScrollView(.vertical, showsIndicators: false) {
                  VStack(spacing: 0) {
                      ForEach(companies) { data in
                          CategoryItem(company: data)
                          
                      }
                  }
              }
              .background(.ultraThinMaterial)
              .navigationBarTitle("Compañías telefónicas")
              
          }
      }
  }

Finalmente..

Importante! Dentro de nuestro archivo principal en mi caso (ProjectViewApp) debe cambiar el ViewController() por HomeView(), Ahora cuando compile su código debería ejecutarse sin problemas. 


      @main
      struct ProjectViewApp: App {
          var body: some Scene {
              WindowGroup {
                  HomeView()
              }
          }
      }
